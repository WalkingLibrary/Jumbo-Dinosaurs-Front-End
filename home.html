Sign Saver Mod

The sign Saver Mod is a Minecraft forge Mod that allows you to save the signs you walk beside in the game to a txt file.

Made by yours truly, the code is open source and can be found here


Install

To install the mod: Download it, unzip it, place the jar inside of ./.minecraft/mods, then Run Minecraft with forge.
