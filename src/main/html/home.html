Book and Sign Saver Mod

    Features:

  • Allows the player to save sign and book data to a json and txt file

  • Allows the player to send their sign and book data to Jumbo Dinosaurs

  • Notifies the player when a Book list or a Sign is saved with either Green or Orange message

    (Orange is to notify the Player that it was sent to Jumbo Dinosaurs and Saved)

    (Green is to notify the Player that it was only Saved and not sent To Jumbo Dinosaurs)

Made by yours truly, the code is open source and can be found here.



How to Use

Once installed and in game press the default Minus key to open the mod's menu. Here you can log into your Jumbo Dinosaurs Account and also change mod settings.

By default the mod does not allow itself to send data to Jumbo Dinosaurs. To change this open the mod menu and toggle the mod's ability to do so.

When walking around the mod will automatically check for signs and save them to ./.minecraft/Jumbo's Mods/Saved Signs/. This can be turn off in the mod's menu.

To save books simply open a chest, enderchest, or shulker box In game and there should be a button in the top right corner to save the contents of that container.

The contents is saved to ./.minecraft/Jumbo's Mods/Book Lists/(the file name you set)(.json/.txt).

You will also have the option to change the name of the file that the book list contents is saved to. If you enter an exsisting file name, the mod will append the contents to the file name set by you and will also check for duplicates and adjust their count accordingly before saving.

To Install

To install the mod: Download a version, unzip it, then place the jar inside of ./.minecraft/mods, then Run Minecraft with forge.

Versions



  • .1.1 For Minecraft 1.12.2
